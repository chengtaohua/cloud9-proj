If you want to be a developer or programmer, you need to be able to do coding. So, we could say it is more important to do actually coding than just thinking. Then, what should we do first? We need to know what programming language is and choose which programming language we want to learn.

What is programming language ?

Computer programming languages allow users to give some instructions to their computers, make computers execute what they want, and finally solve problems using algorithms in computers. There are tons of programming languages including COBOL, C++, Java, or Python.

Also, we could also use other languages they are not considered real programming languages such as HTML, CSS, or SQL. HTML is a markup language, and CSS is a style sheet language. Then, SQL might be considered a data query language or database management language. Because those languages do not contain programming logic, many people do not think they are programming languages. In this article, we are talking about programming languages, so let’s put aside these languages for later.

How to learn Java coding?

You could go to a college and take some programming classes, but it might be still not enough to enhance or practice coding or you need an extra course that is simpler and easier doing just coding.

Learning in a website

Here is good news for you! There are many websites that teach people coding step by step. And here is an article that you might as well read:

Top 10 Websites to Learn Coding (Interactively) Online.

Many of you must have heard about Codecademy or even used that website. You can try some coding of many programming languages on that websites even without installing any applications. Or, if you want to learn web programmings, Treehouse could be a good option.

Taking a course in an online learning institution

You could also take some online courses such as from Udemy or Coursera. Unlike learning on Codecademy, you need to install some applications or programs first. In fact, if you really want to improve your coding skill, you need to install and do coding on your own computer.

What program or application do we need?

If you want to code in Java, you first have to download Java platform (JDK) and install on your computer. Then, you should need Java IDEs (Integrated Development Environment). Here is an article that introduces 10 Java IDEs: 10 Best Java IDEs of All Time. Now, we check two of them, Intellij IDEA and Eclipse. Eclipse is a popular one that is an open source tool, which means it is free. Also, it has a great community of developers. Intellij IDEA is not a free but it has a great feature, especially debugging. In addition, they allow students use it for free during they enroll in a college.

Now, you have everything to do coding in Java. Just do coding!
